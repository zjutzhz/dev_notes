## Windows 下计算 MD5
```cmd
certutil -hashfile filename MD5
certutil -hashfile filename SHA1
certutil -hashfile filename SHA256
```
参考：[Windows下查看文件MD5值](http://blog.csdn.net/xibeichengf/article/details/48750315)
